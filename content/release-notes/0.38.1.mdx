This release is a hotfix to version [0.38.0](/release-notes/0.38.0/).

Contains [#3213](https://github.com/CoinAlpha/hummingbot/pull/3213) a hotfix for Huobi connector fails to track submitted order due to "API signature not valid" as described in GitHub issue [#3197](https://github.com/CoinAlpha/hummingbot/issues/3197).
