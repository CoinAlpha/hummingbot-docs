Gate io is a one stop shop exchange connector. Trade in Markets, Margin, Perpetual, or even Futures.

## Using the Connector

Connect to [Gate IO](https://www.gate.io/ref/4566709) Spot Trading by creating an account and generating an api key.

Input the api key from your Gate io account in the client cli.
